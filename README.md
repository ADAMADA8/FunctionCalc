# FunctionCalc

Простая CLI-программа для вычисления значений функции `f(x)` по шагам.

## Требования

- Rust (stable)
- Cargo

## Запуск

Сборка и запуск:

```bash
cargo run
```

## Использование

После запуска программа попросит:

1. Ввести функцию `f(x)` (например, `x + 10 * 4`).
2. Ввести параметры через пробел в формате:
   `<чему равен первый x> <количество шагов> <размер шага>`

Пример ввода:

```
f(x) = x * x + 1
Параметры: 0 10 1
```

Пример вывода:

```
Шаг | x  | Результат
1   | 0  | 1
2   | 1  | 2
3   | 2  | 5
4   | 3  | 10
5   | 4  | 17
```

## Примечания

- Функции парсятся библиотекой `exmex`.
- Выводится таблица: номер шага, `x`, результат `f(x)`.
